# Реализация IRC сервера на Elixir

###### Реализация спецификации [RFC-1459](http://www.kvirc.ru/wiki/Спецификация_IRC_протокола_(rfc1459)_на_русском_языке)  

Запуск: 
```bash 
mix run --no-halt
```
